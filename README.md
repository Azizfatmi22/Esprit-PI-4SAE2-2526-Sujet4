# 🚀 Esprit-PIDEV-4SAE2-2026-ForminiPlatform

## 📘 Overview

This project was developed as part of the **PIDEV – 4th Year Engineering Program** at **Esprit School of Engineering – Tunisia** (Academic Year 2025–2026).

**Formini Platform** is a modern **AI-powered e-learning system** built using a **Microservices Architecture**, designed to deliver scalable, secure, and personalized learning experiences.

The platform enables:
- Smart course management  
- AI-driven learner evaluation  
- Personalized training paths  
- Secure and modular system design  

---

## 🎯 Features

- 📚 Course and training program management  
- 👤 User and role management (Admin, Learner, Partner)  
- 🤝 Trainer recruitment and validation  
- 🗓️ Smart scheduling system  
- 📝 Exams, evaluation, and certification  
- 💳 Subscription and billing system  
- 💬 Community forum  
- 🤖 AI-based personalization and prediction  

---

## 🧩 Microservices Architecture

Each service is **independent**, **scalable**, and owns its **database**.

- 🔐 Identity & Access Service (Keycloak)
- 👤 User Profile Service  
- 🤝 Trainer Hiring Service  
- 📚 Course Management Service  
- 💳 Subscription & Billing Service  
- 🗓️ Learning Schedule Service  
- 📝 Evaluation & Certification Service  
- 💬 Forum Service  

---

## 🤖 AI Integration

| AI Model | Purpose |
|--------|--------|
| Supervised Learning | Predict learner success |
| Unsupervised Learning | Cluster learners by behavior |
| Reinforcement Learning | Recommend personalized learning paths |

---

## 🏗️ Architecture

- API Gateway  
- Eureka Service Discovery  
- Centralized Configuration  
- Database per microservice  

---

## 👥 Actors

| Actor | Description |
|------|------------|
| Administrator | Manages users, courses, and system |
| Learner | Enrolls, learns, and gets certified |
| Partner | Provides trainers and manages requests |

## 🚀 Getting Started

### Prerequisites
- Java 17+  
- Node.js  
- Angular CLI  
- Docker (optional)
  
### Run Frontend
    ng serve
### Run Backend
    mvn spring-boot:run

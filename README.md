# 🚀 Esprit-PIDEV-4SAE2-2026-ForminiPlatform

## 📘 Overview

This project was developed as part of the **PIDEV – 4th Year Engineering Program** at **Esprit School of Engineering – Tunisia** (Academic Year 2025–2026).

**Formini Platform** is an intelligent AI-powered e-learning and professional certification platform built using a **Microservices Architecture**.  
The platform aims to modernize digital education by providing scalable, secure, adaptive, and interactive learning experiences for both individuals (**B2C**) and enterprises (**B2B**).

Formini combines:
- Intelligent learning management
- AI-driven assessments
- Hybrid training management
- Interactive live learning
- Gamification and certification
- Smart scheduling and personalization
